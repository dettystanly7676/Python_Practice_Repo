-> to create a new field in a data set 
	create calculated field -> type on the equation -> OK
	calculated field will be visible in the field section on left side
	when it is dragged in a new sheet the va;ues will be zero
	to change that,
	select the field -> right click -> format -> go to the top left corner -> click on specific options(AGG, SUM ..) -> change the numbers to your convienent format
	
creating a hierarchy
	-> click on a field -> down arrow -> hierarchy -> create hierarchy
	-> click on a field to add to the created hierarchy -> down arrow -> hierarchy -> add to hierarchy -> hierarchy_name
